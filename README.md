# todo-list

## 使用到的 packages

1. react router
2. tailwindcss
3. shadcn/ui

## 技術說明

1. 試著使用 typescript 體驗嚴格的變數型態
2. 使用 shadcn/ui，避免花時間寫弄元件及 css
3. 使用 context 管理 todo 狀態
4. todos 儲存在 lcoalstorage
5. 初次載入時使用 useEffect 從 lcoalstorage 載入 todos
6. 使用 ref 處理輸入值
7. 使用 react router 來呈現完成與否的 todos
8. 目前只支援新增/刪除/更改完成狀態
